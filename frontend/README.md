# Veritas Frontend

Aplicação Next.js para interface do Veritas.

## Instalação

```bash
npm install
```

## Executar

```bash
npm run dev
```

Aplicação disponível em http://localhost:3000

## Build

```bash
npm run build
npm start
```

## Estrutura

```
frontend/
└── src/
    └── app/
        ├── page.tsx         # Home
        ├── analyze/         # Página de análise
        └── about/           # Sobre
```

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
